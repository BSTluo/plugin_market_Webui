> 可以将图片收藏到不同的图包！

## 功能
- 用于给蔷薇表情包提供分类功能，跨端同步由猹猹完成，有问题和改进建议欢迎寻找作者反馈

## 使用说明
手机端长按图包里的表情，电脑端右键

跨端同步功能：如果你有A设备和B设备，想要把A设备的分类移到B设备上，就需要在B设备上请求同步，在A设备上同意。也不一定是不同的设备，不同的浏览器等都可

允许用户通过文本导入导出表情包分类数据，也就是你可以导入别人的表情包，也可以把自己的表情包分享给别人

注意：清除浏览器数据会导致分类数据也被清除
